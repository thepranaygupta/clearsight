# ClearSight Documentation

Welcome to the ClearSight documentation. ClearSight is an AI-powered WCAG 2.1 accessibility checker that crawls your entire site and delivers actionable results.

## Quick Start

1. Go to the [Dashboard](/dashboard)
2. Enter your site URL
3. Click **Start scanning** — ClearSight will crawl every page
4. Review your results with AI-generated fix suggestions

## What ClearSight Does

- **Full-site crawling** — discovers every page via BFS link following + sitemap parsing
- **WCAG 2.1 A & AA checks** — 50+ axe-core rules plus custom engines
- **AI enrichment** — human-readable descriptions and fix suggestions for every issue
- **Issue tracking** — track new, fixed, and recurring issues across crawls
- **Export** — PDF reports with executive summaries and Excel spreadsheets
