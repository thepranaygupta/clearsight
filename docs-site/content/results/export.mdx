# Export Reports

Export scan results as PDF or Excel for sharing with stakeholders.

## PDF Report

A formatted document with:

- **Executive summary** — AI-generated narrative overview
- **Score gauge** and key metrics
- **Issue breakdown** by severity
- **Full issue list** with descriptions, selectors, and fix suggestions
- **Page screenshot** with issue highlights

## Excel Spreadsheet

A multi-sheet workbook with:

- **Summary sheet** — score, scan metadata, top priorities
- **Issues sheet** — filterable table of all issues with every field
- **Positive findings** — what the page does well

## How to Export

### Single-Page Scan

From the scan results page, click the **Export** dropdown and choose PDF or Excel.

### API

```bash
# PDF
curl -o report.pdf \
  "http://localhost:3000/api/scans/{scanId}/export?format=pdf"

# Excel
curl -o report.xlsx \
  "http://localhost:3000/api/scans/{scanId}/export?format=excel"
```

## Notes

- Export is only available for completed scans (`completed` or `completed_partial`)
- The PDF executive summary is generated by AI on-the-fly — there may be a brief delay
- Excel files include all issue fields for maximum filterability
